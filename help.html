WARNING PBLV1 SHOULD NOT BE INSTALLED AS THE SOFTWARE "PBLV1" IS EXPERIMENTAL AND CAN DAMAGE HARDWARE AND HAS NO INTERNET FOR MORE INFORMATION GO TO PBL MAIN PAGE for more support. what is a bootable storage device?
A bootable storage device is a storage device that the machine can boot from. Examples: USB drives SD cards Floppy disks SSDs Hard drives CDs DVDs Etc.
How to boot onto BIOS or UEFI? first power off your computer. Second repeat pressing the boot key until you get to the boot menu (BIOS or UEFI) common keys for boot menu Dell: F2 (or F12 for boot menu) HP: Esc or F10 Lenovo: F2 or Fn+F2 (or Novo button) ASUS: F2 or Del Acer: F2 or Del MSI: Del
